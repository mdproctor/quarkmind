IntentQueue — thread-safe buffer. SC2 frame loop writes to CaseFile; CaseEngine reads CaseFile and writes intents; CommandDispatcher drains intents each frame. Neither loop ever blocks the other.
CaseFile — the shared blackboard. GameStateTranslator pushes SC2 observations in; TaskDefinitions read and write reasoning state; changes trigger CaseEngine's next reasoning cycle.
